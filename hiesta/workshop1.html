Sebelumnya kalian tau tidak, bahwa pembuatan robot membutuhkan alat sensor yang memiliki berbagai fungsi? Ternyata ada banyak sekali jenis sensor yang bisa ditanamkan pada robot loh! Diantaranya adalah:
1. Touch Sensor merupakan jenis sensor yang akan mendeteksi ketika disentuh. Pada robot, touch sensor digunakan untuk mendeteksi objek yang ada pada tangan robot, mencegah terjadinya tabrakan pada robot beroda, dan masih banyak lagi. Contoh touch sensor yang paling sederhana adalah Push Button.
2. Light Sensor Sensor ini mendeteksi cahaya atau peka terhadap cahaya disekitarnya. Dengan sensor ini robot dapat mengetahui gelap dan terang suatu objek, tempat, siang atau malam. Untuk menentukan gelap dan terang suatu tempat biasa menggunakan LDR Sensor, sementara untuk keperluan Robot Pengikut Garis (Line Follower) menggunakan InfraRed Sensor.
3. Balance Sensor Biasa digunakan untuk membuat robot tetap seimbang. Mengetahui kemiringan, dan membantu bangun saat robot terjatuh. Salah satu contohnya adalah Gyroscope, dipakai juga pada Smartphone. Wah ternyata banyak sekali kan sensor yang dapat digunakan pada robot. Untuk mengetahui lebih banyak lagi tentang sensor ataupun pembuatan robot, ikuti workshop HIESTA 2018 yuk!
